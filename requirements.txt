This app mainlt tracks the company fanancials and user portfolio datils. This will have past variations 
of the securities prices for.a longer period of time. Moomoo platform is a good example of the type of app 
I want to build. The main focus of this app will be on the financial data of the companies and the portfolio 
management of the users. The app will have a simple and user-friendly interface, and will be accessible on 
both web and mobile platforms.

Users:
All users will extend from the base user model.
- Admin
- Regular User
- Feeders

Features:
1. Authentication:
1.1 User can register and login using email and password.
1.2 User can also login using Google auth API.
1.3 Password reset functionality will be implemented.
1.4 Verify email functionality will be implemented.
1.5 Password reset functionality will be implemented.
1.6 Delete account functionality will be implemented.
1.7 Admin can manage users, such as activating/deactivating accounts, assigning roles, etc.

2. User Management:
2.1 User can view their profile and update their information.
2.2 Users can invite other users to join the platform.

3. User financeal data management:
3.1 Feeders can add and update the financial data of the companies
3.2 Users can read stock prices in real-time and store them locally for a longer period of time
3.3 Users can view their portfolio and the financial data of the companies they are invested in
3.4 Users can view the historical data of the stock prices and financial data of the companies
3.5 Users can view financial parameters of their portfolio, such as today P&L, total P&L, etc.
3.6 Users can add to favorite companies.

4. Securities information management:
4.1 Feeders can add and update the company financials data.
